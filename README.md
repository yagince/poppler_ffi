# Poppler

Ruby bindings for Poppler

## Requirements

- libpoppler-glib-dev >= 0.38

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'poppler-ruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install poppler-ruby
